Java-api-to-mtgdb

A Java API to access the web services provided by mtgdb.info.

Download .zip Download .tar.gz View on GitHub

Welcome.

This is the homepage for the Java API to the mtgdb.info database. The goal of this project is to provide a clean, Java approach to accessing the great features available at mtgdb.info.

The jar

If you want to get right to the point, you can download the jar and just add it to your project to start using the API.

Javadoc

If you'd like to check out what the Java API offers without actually checking out the code, the Javadoc is available here:
